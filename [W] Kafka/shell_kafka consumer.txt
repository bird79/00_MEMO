# kafka consumer shell
## 옵션 몰라
두번째거는 암것도 안되네
```sh
./bin/kafka-topics --zookeeper localhost:2181 --alter --topic A1 --config retention.ms=1000
./bin/kafka-topics --zookeeper localhost:2181 --alter --topic A1 --deleteConfig retention.ms
```


## 이거하면 없어지는거 같어
위에서 옵션을 정해줘서 그런가?
```sh
./bin/kafka-console-consumer --zookeeper localhost:2181 --topic A1 --from-beginning
```
